Welcome to M-nvm

M-nvm is a fork from the original nvm, which does not work on Windows platform. M-nvm makes a minimalist's change to the original nvm code so that you can easily manage your nodejs versions from a bash prompt in Windows.

Currently only MinGW32 works.

Usage

  1. clone this repository
  2. source the nvm.sh file
  3. begin nvm-ing ;)
